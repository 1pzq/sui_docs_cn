在您能够创建您的第一个dApp之前，您必须已经[安装了Sui](./getting-started/sui-install.mdx)。

您使用Move编写在链上存在的软件包，这意味着它们存在于您发布它们的Sui网络上。本节中的说明将引导您完成编写基本软件包、调试和测试代码以及发布的过程。您需要按照这些说明顺序完成练习。

某些说明中您将使用`move` Sui CLI 命令。Sui CLI是与二进制文件一起安装的，因此如果您按照安装说明进行了安装，它就已经在您的系统上。为了验证您已经安装了它，请在终端或控制台中运行以下命令。

```shell
sui --version
```

如果控制台没有回应类似于以下版本号的内容，请参阅安装Sui的说明。

```shell
sui 1.xx.x-abc123xyz
```
