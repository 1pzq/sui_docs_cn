安装 Sui 的最快方式是使用每个版本附带的二进制文件。如果您需要更多对安装过程的控制，可以从源代码进行安装。为了充分利用容器化，您可以使用 sui 存储库的 `docker` 文件夹中的 Docker 镜像。

## 支持的操作系统 {#supported-operating-systems}

Sui 支持以下操作系统：

- Linux - Ubuntu 版本 20.04(Bionic Beaver)
- macOS - macOS Monterey
- Microsoft Windows - Windows 10 和 11

## 使用 Homebrew 安装 {#install-homebrew}

如果您使用 [Homebrew](https://brew.sh/)，您可以使用以下命令安装 Sui：

```bash
brew tap mystenlabs/tap
brew install mystenlabs/tap/sui
```

## 从二进制文件安装 {#install-binaries}

每个 Sui 版本都提供了一组适用于多个操作系统的二进制文件。您可以从 GitHub 下载这些二进制文件并使用它们安装 Sui。

1. 转到 https://github.com/MystenLabs/sui。
1. 在右侧窗格中，找到 **Releases** 部分。

    ![GitHub 中的 Sui 发行版](./images/releases.png)
1. 单击标记为 **Latest** 的发行版，打开发行版的页面。
1. 在发行版的 **Assets** 部分，选择与您的操作系统相对应的 .tgz 压缩文件。
1. 双击下载的文件。如果文件没有自动展开，请手动解压文件。
1. 打开展开的文件夹，双击适当的二进制文件进行安装，以 sui-`<OS>`-`<ARCHITECTURE>` 开头：

    - sui-faucet-`<OS>`-`<ARCHITECTURE>`: 本地龙头，用于在本地网络上铸币。
    - sui-indexer-`<OS>`-`<ARCHITECTURE>`: 用于本地 Sui 网络的索引器。
    - sui-`<OS>`-`<ARCHITECTURE>`: 主 Sui 二进制文件。
    - sui-node-`<OS>`-`<ARCHITECTURE>`: 运行本地节点。
    - sui-test-validator-`<OS>`-`<ARCHITECTURE>`: 在本地网络上运行开发的测试验证器。
    - sui-tool-`<OS>`-`<ARCHITECTURE>`: 为 Sui 提供实用工具。

    Sui 二进制文件需要一些时间来下载和安装文件，因此请确保您为其完成分配足够的时间。实际完成时间取决于您的网络连接和计算机规格。

### 确认安装 {#confirm-the-installation}

要确认 Sui 安装正确，请键入 `sui` 并按 Enter 键。您应该看到有关已安装的 Sui 版本的消息以及有关使用 Sui 命令的帮助信息。

## 从源代码安装

按照本主题中的说明安装与 Sui 网络交互所需的 Rust crates（包），包括 Sui CLI。

要从源代码安装 Sui，首先需要为您的操作系统安装其[先决条件](#prerequisites)。在安装支持技术之后，您可以从[源代码安装 Sui 二进制文件](#install-sui-binaries-from-source)。

您还可以下载[源代码](./sui-environment)以本地访问文件。

## 先决条件 {#prerequisites}

您的系统需要以下先决条件才能成功安装 Sui。

### Rust 和 Cargo {#rust-and-cargo}

Sui 在所有支持的操作系统上都需要 Rust 和 Cargo（Rust 的包管理器）。安装 Rust 的建议方法是使用 `rustup` 并使用 cURL。

安装说明中的一些其他命令还需要运行 cURL。如果无法运行 cURL 命令来安装 Rust，请在安装 Rust 之前查看以下部分中有关为您的操作系统安装 cURL 的说明。

在 macOS 或 Linux 上使用以下命令安装 Rust 和 Cargo：

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

如果您使用的是 Windows 11，请查看[Rust安装程序](https://www.rust-lang.org/tools/install)上 Rust 网站的相关信息。安装程序会检查是否存在 C++ 构建工具，并在必要时提示您安装它们。选择最符合您环境的选项，并按照安装向导中的说明操作。

有关其他安装选项，请参阅[安装 Rust](https://www.rust-lang.org/tools/install)。

Sui 使用最新版本的 Cargo 来构建和管理依赖关系。有关更多信息，请参阅 Rust 网站上的[Cargo安装](https://doc.rust-lang.org/cargo/getting-started/installation.html)页面。

使用以下命令使用 `rustup` 更新 Rust：

```bash
rustup update stable
```

## Install Sui binaries from source {#install-sui-binaries-from-source}

Run the following command to install Sui binaries from the `testnet` branch:

```shell
cargo install --locked --git https://github.com/MystenLabs/sui.git --branch testnet sui
```

The install process can take a while to complete. You can monitor installation progress in the terminal. If you encounter an error, make sure to install the latest version of all prerequisites and then try the command again.

To update to the latest stable version of Rust:

```shell
rustup update stable
```

The command installs Sui components in the `~/.cargo/bin` folder.

## Upgrade Sui binaries

If you previously installed the Sui binaries, you can update them to the most recent release with the same command you used to install them:

```shell
cargo install --locked --git https://github.com/MystenLabs/sui.git --branch devnet sui
```
