title: 获取 Sui 地址
description: 在开始交易 NFT、购买 SUI 代币或执行其他交易之前，你需要在 Sui 网络上拥有一个地址。
---

地址是在 Sui 区块链网络上唯一且匿名标识帐户的方式。换句话说，地址是用户在 Sui 网络上存储和使用代币的一种方式，而无需提供任何个人身份信息（例如电子邮件地址、电话号码等）。例如，如果你想购买一些 SUI 代币来玩游戏，你必须指定一个地址，这些代币将被存入该地址。

Sui 地址是唯一的，类似于社会安全号码或个人身份号码对一个人的唯一性。然而，在 Sui 中，你可以创建和拥有多个地址，所有这些地址都是唯一的。

在 Sui 中，一个地址是 32 字节，通常以带有 `0x` 前缀的 base58 编码。例如，这是一个有效的 Sui 地址：`0x02a212de6a9dfa3a69e22387acfbafbb1a9e591bd9d636e7895dcfc8de05f331`。你可以使用 [Sui Explorer](https://suiexplorer.com/) 网站查找有关[此地址](https://suiexplorer.com/address/0x02a212de6a9dfa3a69e22387acfbafbb1a9e591bd9d636e7895dcfc8de05f331)及其拥有的对象的更多信息。

如果你想了解 Sui 地址是如何从私钥派生出来的以及其他与密码学相关的主题，请参阅 [Keys and Addresses](/concepts/cryptography/transaction-auth/keys-addresses.mdx) 主题。

## 如何获取 Sui 地址

Sui 提供多种获取 Sui 地址的方式。以下是两种最常见的方法。

### Sui 钱包

对于首次使用者而言，通过 [Sui 钱包 Chrome 浏览器扩展](https://chrome.google.com/webstore/detail/sui-wallet/opcgpfmipidbgpenhmajoajpbobppdil) 是获取 Sui 地址的一种直接方式。安装扩展后，有几种方法可以创建地址。

打开 Chrome 中的 Sui 钱包浏览器扩展，然后：
- 使用你的 gmail/twitch/facebook 账户 (ZkLogin) 并按照屏幕上的说明操作
- 单击 **More Options** → **Create a new passphrase account**。然后按照屏幕上的说明操作。

有关 Sui 钱包及如何确保其安全性的更多信息，请参阅 [Sui 钱包文档](https://mysten-docs-site.vercel.app/wallet)。

### 命令行界面

如果你使用 Sui 命令行界面 (CLI) 与 Sui 网络交互，可以使用 `sui client` 命令生成新地址。默认情况下，当 Sui CLI 首次运行时，它将提示你设置本地钱包，然后生成一个 Sui 地址和关联的秘密恢复短语。确保你记下秘密恢复短语并将其存放在安全的地方。

要生成新的 Sui 地址，请使用 `sui client new-address ed25519`，其中指定了 keypair scheme 标志的类型为 `ed25519`。

有关更多信息，请参阅 [Sui Client CLI](../../../references/cli/client.mdx) 文档。

要查看本地计算机上本地钱包中生成的所有地址，请运行 `sui keytool list`。有关 keytool 选项的更多信息，请参阅 [Sui Keytool CLI](../../../references/cli/keytool.mdx) 文档。

:::danger

与 Sui 地址关联的私钥存储在 CLI 安装的计算机上的本地位置，即 `~/.sui/sui_config/sui.keystore` 文件中。确保不要将其暴露给任何人，因为他们可以使用它来访问你的帐户。

:::
