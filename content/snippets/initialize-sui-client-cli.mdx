在发布代码之前，如果尚未初始化 Sui Client CLI，你必须先进行初始化。为此，请在项目的根目录下的终端或控制台中输入 `sui client`。如果你收到以下响应，请完成剩余的说明：

```
Config file ["<FILE-PATH>/.sui/sui_config/client.yaml"] doesn't exist, do you want to connect to a Sui Full node server [y/N]?
```

输入 `y` 以继续。你将收到以下响应：

```
Sui Full node server URL (Defaults to Sui Devnet if not specified) :
```

保留此项为空（按 Enter 键）。你将收到以下响应：

```
Select key scheme to generate keypair (0 for ed25519, 1 for secp256k1, 2: for secp256r1):
```

选择 `0`。现在你应该已经设置了 Sui 地址。
