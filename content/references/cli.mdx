Sui提供了命令行工具箱，用于与Sui网络、其功能和Move编程语言进行交互。 包括 Sui Client CLI， Sui Console CLI， Sui Keytool CLI， Sui Move CLI， 以及 Sui Validator CLI。

{@include: ../snippets/cli-check-install.mdx}

## 更新 CLI

想要获取CLI的最新版本，您可以从终端或控制台运行以下命令。请确保将`<BRANCH-NAME>`替换为`main`、`devnet`、`testnet`或`mainnet`以获取所需版本。有关可用分支的更多信息，请参阅[Sui环境设置](/guides/developer/getting-started/sui-environment.mdx)。

```shell
cargo install --locked --git https://github.com/MystenLabs/sui.git --branch <BRANCH-NAME> sui
```

## Sui CLI 命令

Sui CLI 包括以下五种顶级命令，对于尚未记录的命令，请使用help标志。例如，`sui validator --help`。

- **[Sui Client CLI](./cli/client.mdx):** 使用 `sui client` 命令与Sui网络交互。
- **[Sui Console CLI](./cli/console.mdx):** 使用 `sui console` 打开一个与当前活动网络交互的交互式控制台。
- **[Sui Keytool CLI](./cli/keytool.mdx):** 使用 `sui keytool` 命令访问密码工具。
- **[Sui Move CLI](./cli/move.mdx):** 使用 `sui move` 命令执行与Move编程语言相关的命令。
- **Sui Validator CLI:** 使用 `sui validator` 命令访问对Sui验证节点有用的工具。
